How It Works

Six intelligent processing steps run automatically on every upload.

01
🗑️

Remove Invalid Rows

Any row with a blank CE Commodity Description is automatically removed before processing begins.

02
✂️

Split Description

Manifested Description is split on / into three parts: MID Code, HS Code, and Product Description.

03
🧬

Clean MID Codes

Embedded numbers are stripped intelligently. PKUMANAESIA51310 becomes PKUMANAESIA.

04
🔄

Recover HS Codes

Missing HS codes are automatically recovered from the CE Item HSCode column as a fallback.

05
✨

Format Output

Final format is always MIDCODE/HSCODE/DESCRIPTION — no extra spaces, no duplicate slashes.

06
💾

Excel Export

The cleaned file is exported as a formatted .xlsx with all original columns preserved.

Output
PKUMANAESIA/6204120010/MEN SUIT M/O 100
